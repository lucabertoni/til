# Scrollare il file
Quando si necessita di scrollare un file si possono usare i tasti `pag-su` e `pag-giu`. Esistono per√≤ due combinazioni di tasti alternative.
Scroll verso il basso:
```
C-v
```
`C` si riferisce al tasto `ctrl`.

Scroll verso l'alto:
```
M-v
```
`M` si riferisce al tasto `alt`.

I due comandi solitamente non muovono la posizione del cursore.
